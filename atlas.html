Atlas

Atlas is a distributed, web-based application for the discovery,
visualization, and analysis of multi-dimensional geospatial data.

Problem

Current data visualization tools do not offer effective design solutions for usability issues posed by the complex relationship between spatial and numeric data.

Objective

Create a design element or elements that allow users to alter various features of appearance and time series in a settings panel.

Role

User Interface Designer

Process

First

Determine what filtering abilities users will need to alter the visual representation of the data.

Second

SKETCH. A co-worker on this project had previously come up with a few sketches that were able to prompt more sketches of what a filtering design element could look like.

Third

Create a digital design in Axure that can be used for user testing. The final design was not exactly like any of the sketches that had previously been done, but the sketches were a jumping off point for creating a usable filtering element that contains all the necessary filtering options.
